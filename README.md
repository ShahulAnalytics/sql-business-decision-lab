# SQL for Business Decision Making

## Project Overview
This repository presents structured SQL case studies focused on solving real-world business problems. Each case simulates practical analytics scenarios faced by organizations and demonstrates how SQL is used to extract insights for decision-making.

## Objectives
- Apply SQL to business-focused data problems
- Perform KPI analysis using real-world scenarios
- Demonstrate joins, aggregations, and window functions
- Translate query outputs into business insights

## Case Study Structure
Each case study includes:
- Business Problem
- Data Question
- SQL Query
- Business Insight
